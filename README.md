# Prac2Time
 Stuff I found cool and wanted to try out in Unreal!


 Controls
 WASD - to move
 G - Grapple Hook
 Space on ledges to vault
 B - Drone
  WASD - Move Drone
  Q and E - Lift or Drop Drone
 I - Inventory
 Hold R - Pick up Weapon
 Q - Airstrike Screen
